Regions.js

A polyfill for the CSS3-Regions

It's all about layout baby !

So you want to create some complex, magazine like, layouts !

You know what ? That's possible ! Yeah ! Really ! The only thing you must do is the following: Just put some cool CSS as described in the W3C's specification and you're good to go ! Yeah that's really easy and you can get some really impressive results !

But ... !

But what ? Oh yeah, I forgot that the only existing implementation is in a prototypal webkit browser made by Adobe, you can't ask to your customers to use this, of course. So what can you do ?

Regions.js appears to be the only solution against this problem for the moment. Ok, it's in beta. Ok, you may need to write 2 lines of javascript (We're trying to fix this last problem). And that's terribly slow ! But seriously ! Do you want to create really amazing layouts or not ? (Like the one on this page for example !)

Amazing ! How can I use it ?

As the API might change in a close future, you should check it on the github repository.

You might need to use jQuery and Lettering.js (that are excellent tools by the way !) in order to use it. You'll also get better results with Modernizr.
